# Extension Points

## Root control plane

- `rendo.template.json`
- `rendo.project.json`
- `docs/*`

## `__RENDO_CORE_LABEL__` core surface

- `__RENDO_CORE_DIRECTORY__/README.md`
- future files introduced by a base template that inherits this core

## Rule

The core layer should define how a `__RENDO_CORE_LABEL_LOWER__` grows, not what product or vendor it becomes.
