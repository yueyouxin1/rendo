# Template Assets

Rendo template assets are layered as:

1. `core-starter`
2. `base/<template-kind>/...`
3. `derived/<template-kind>/...`

Current implemented template kind:

- `starter`

Current role distribution:

- `core-starter`: unique foundation
- `base/starter/*`: official canonical starter references
- `derived/starter/*`: starter assets derived from a base starter
