# __RENDO_PROJECT_NAME__

This project was created from the Rendo `application-base-starter`.

## What it validates

- The official Application Base Starter Template is multi-surface-capable
- `rendo create application --surfaces web` generates only the web surface
- `rendo create application --surfaces web,miniapp` or `web,mobile` keeps a valid multi-surface structure
- The web surface remains runnable even when extra surface placeholders are generated

## Commands

```bash
npm run web:install
npm run web:dev
npm run web:check
npm run docker:up
```
