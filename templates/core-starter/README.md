# __RENDO_PROJECT_NAME__

This project was initialized from the Rendo `core-starter`.

## What it is

- A runnable, headless-first engineering substrate
- A contract surface for manifests, providers, packs, and agent instructions
- A safe place to start building a new Domain Starter

## What it is not

- Not a Next.js application
- Not a product UI
- Not a hard dependency on Rendo Cloud

## Commands

```bash
pnpm install
pnpm dev
pnpm health
pnpm check
pnpm docker:up
```

## Runtime

- HTTP health endpoint: `GET /health`
- Default port: `3210`
- Default provider mode: `local-mock`
