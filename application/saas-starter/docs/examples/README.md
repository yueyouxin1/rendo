# 配置参考样例

这些文件从旧的 参考目录迁移而来，只保留为：

- 当前独立工作区内的讨论素材
- 本机已验证过的镜像与版本参考
- 后续正式模板实现前的临时对照样例

它们不是当前模板的 source of truth，也不会自动覆盖：

- `ops/docker/compose.yaml`
- `src/apps/web/Dockerfile`
- 根 `package.json`

使用原则：

1. 先以当前工作区真实代码和 `ops/` 为准。
2. 只有在需要对照本机已验证镜像标签或脚本命名时，才阅读这些样例。
3. 如果正式实现吸收了这些样例中的内容，应把结果写回正式位置，而不是长期依赖“仅供参考”文件。
